---
  根本原因                                                                                                                                                                                                                                                       
          
  Connection pool is full 来自 urllib3，说明是同步 requests 库（而非 httpx）在某处触发了。结合服务地址 steins-middleware.steins.net，这指向 Nacos / 中间件 SDK 的连接，而不是 embedding 或 VLM 服务。                                                            
                                                                                                                                                                                                                                                                 
  真正的问题链分为两层：                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                 
---
  第一层：并发爆炸（触发原因）
                              
  位置：rag/scheduler/processor.py:271-280
                                                                                                                                                                                                                                                                 
  text_embeddings, hybrid_embeddings = await asyncio.gather(
      vl_embedding_model.aembed_list(text_embedding_data_list, ...),                                                                                                                                                                                             
      vl_embedding_model.aembed_list(hybrid_embedding_data_list, ...),                                                                                                                                                                                           
  )
                                                                                                                                                                                                                                                                 
  位置：steins_ai/embeddings/qwen3_vl_embedding.py:346-357（在 third_party/steins-ai 里）                                                                                                                                                                        

  # aembed_list 内部                                                                                                                                                                                                                                             
  tasks = [self.aembed(data, ...) for data in embedding_data_list]
  embed_list = await asyncio.gather(*tasks)  # ← 无并发限制！                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                 
  一个 100 页文档 → 一次性发出 200 个并发 HTTP 请求（text + hybrid 各 100 个）。                                                                                                                                                                                 
                                                                                                                                                                                                                                                                 
---
  第二层：连接池无配置（放大原因）
                                  
  三处 httpx 客户端均使用默认值，没有配置 limits：
                                                                                                                                                                                                                                                                 
  ┌────────────────────────────────────────────┬─────────┐                                                                                                                                                                                                       
  │                    文件                    │  行号   │                                                                                                                                                                                                       
  ├────────────────────────────────────────────┼─────────┤                                                                                                                                                                                                       
  │ steins_ai/embeddings/qwen3_vl_embedding.py │ 103-104 │
  ├────────────────────────────────────────────┼─────────┤
  │ steins_ai/rerankers/qwen_vl_reranker.py    │ 140-141 │
  ├────────────────────────────────────────────┼─────────┤                                                                                                                                                                                                       
  │ camel/models/openai_compatible_model.py    │ 140-170 │
  └────────────────────────────────────────────┴─────────┘                                                                                                                                                                                                       
                  
  httpx 默认 max_connections=100，200 个并发请求超出后开始丢弃连接，底层 urllib3 打出警告日志。                                                                                                                                                                  

---
  修复方案        
          
  优先级最高：给 aembed_list 加 Semaphore
                                                                                                                                                                                                                                                                 
  在 third_party/steins-ai/steins_ai/embeddings/qwen3_vl_embedding.py 的 aembed_list 里加并发控制：                                                                                                                                                              
                                                                                                                                                                                                                                                                 
  async def aembed_list(self, embedding_data_list, max_concurrent: int = 16, ...):                                                                                                                                                                               
      semaphore = asyncio.Semaphore(max_concurrent)                                                                                                                                                                                                              
      async def _limited(data):
          async with semaphore:                                                                                                                                                                                                                                  
              return await self.aembed(data, ...)
      return await asyncio.gather(*[_limited(d) for d in embedding_data_list])                                                                                                                                                                                   
                                                                                                                                                                                                                                                                 
  同时：给 httpx 客户端配置连接池上限
                                                                                                                                                                                                                                                                 
  from httpx import AsyncClient, Limits

  limits = Limits(max_connections=20, max_keepalive_connections=10)                                                                                                                                                                                              
  self._httpx_async_client = AsyncClient(timeout=timeout, limits=limits)
