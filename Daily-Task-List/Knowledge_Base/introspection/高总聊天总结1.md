# 从"被动执行者"到"主动思考者" · 行动手册

> 来源：与老板的一次谈话。核心不是"你技术不行"，而是"你有技术，但还没看懂自己的工作在项目里是怎么创造价值的"。

------

## 〇、先把心态钉住（你已经悟到的，再补两句）

- **老板愿意讲 = 投资，不是客套。** 他花时间讲这些，是因为他对你的预期天花板比你现在的位置更高，不希望你停在"不高不低"的状态。
- **容忍度是一种有限资源，会被消耗。** 这次的反馈是"提醒"，不是"惩罚"。但同一个问题下次再犯，消耗的就是信任额度。目标不是"以后不犯错"，而是"不犯同一个错"。
- **别自己先把自己框死在岗位里。** 他给你画的圈，比你现在站的地方大。

------

## 一、接到任务时：30 秒自问清单 —— 治"被别人安排"

每次秀韬/老板派活，**先别急着开干**，在心里过四个问题：

1. 这个任务在整个项目流程里处在哪一环？**上游是谁、下游是谁**？
2. 他为什么在「这个时间点」把「这个活」派给「我」？（背后通常有个项目层面的原因）
3. 这真的是当前**最该干**的吗？有没有优先级更高的？——如果你判断有，**主动说出来**。
4. 做完它，对项目整体**推进了什么**？

**判断标准：**

- 四个能答出三个 → 你在主动思考。
- 一个都答不上 → 你还在"打螺丝"模式，说明你对全局观察得不够，该去补观察。

------

## 二、沟通时：三个固定动作 —— 治"没听懂就开讲"

> 这次的具体问题：老板说要聊"客户需求"，你听成了别的，还按自己的理解往下讲；他重申后你也没确认，继续按自己思路开讲。

**固定动作（成本极低，收益极高）：**

1. **接收 → 复述确认**：「您是想让我聊 X，对吗？」一句话，能挡住 90% 的跑偏。
2. **不懂 → 直接问**：「这块我不太确定，您指的是…还是…？」知之为知之，**不懂装懂的代价远大于多问一句**。
3. **回应前 → 先答他问的，再延伸**：不要用"你想讲的"替换掉"他问的"。

------

## 三、日常：观察什么 —— 治"听得少、看得少"

工作之余，刻意观察这些（不用全做，从你能接触到的开始）：

- 老板/秀涛跟**客户**聊天时，他们聊什么、**为什么聊这个不聊那个**？
- 同一件事，老板找己居谈和找心怡谈，**方式有什么不同**？为什么？
- 你的工作交到下游（吉君、心怡）手里，给他们带来的是**省力**还是**添乱**？

**路径：观察 → 思考 → 提建议。** 你的判断不一定对，没关系——这个过程本身就在帮你攒"事物运行逻辑"的理解。攒够了，你自然会过渡到"能主动提出有价值建议"的阶段。

------

## 四、自检标尺：你这周更偏哪一栏？

| 维度      | 被动执行者            | 主动思考者                 |
| --------- | --------------------- | -------------------------- |
| 接活      | 等着被派              | 知道什么该先干，会主动认领 |
| 对 leader | 增加他的指挥/监督负担 | 替他分忧、给团队减负       |
| 成长      | 锁死在岗位内          | 向上突破                   |

> 每周五花 5 分钟，回看这周自己更偏哪一栏。

------

## 五、流水线类比（记住这个，能看清职业天花板）

| 角色 | 核心能力                           | 关键转变                       |
| ---- | ---------------------------------- | ------------------------------ |
| 工人 | 把螺丝打快                         | ——                             |
| 班长 | 管理、调度、化解矛盾、对指标负责   | **不是因为螺丝打得快才当班长** |
| 厂长 | 自己做决策（打 A 还是打 B 更赚钱） | 没人告诉你该产多少             |
| 老板 | 定方向、找人、找钱、找客户         | 连"做什么"都要自己定           |

**含义**：如果你的注意力全在"怎么把螺丝打得更快"，反而会离管理者越来越远，因为那是**完全不同的能力**。 **杠杆**：一个人疯狂打螺丝，上限是 8–10 小时体力；但如果你能改进一个团队的管理方式，让 100 个人每天都快 10%，这一个小改动创造的价值远超你拼命干。

------

## 六、接下来 4 周的小练习（别贪多，一周只练一件）

- **第 1 周**：只练"复述确认"。每次接任务，逼自己说一句「我理解您是要…对吗」。
- **第 2 周**：每次接任务，在心里答第一节的 4 个问题，答不上的记下来。
- **第 3 周**：挑 1 次机会，主动问秀涛「为什么这个时间点做这个」，观察他的回答。
- **第 4 周**：尝试就某件事，**主动提 1 条小建议**（哪怕不被采纳）。

------

*一句话总结：先观察，再思考，最后提建议。别让自己的成长，被锁死在"把螺丝打快"这一件事上。*
