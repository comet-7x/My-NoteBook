## HTML结构速览
- **文档声明** (`<!DOCTYPE html>`)：告诉浏览器这是一个现代的 HTML5 网页。
- `<html>`（树根）
    
    - 📄 `<head>`（网页的幕后大脑）
        
        - 🏷️ `<meta>`：设置编码（如 `charset="UTF-8"`，防止中文乱码）。
            
        - 🏷️ `<title>`：标签页的名字。
            
        - 🔗 `<link>`：把写好的 CSS 样式表像外挂一样引进来。
            
    - 🎨 `<body>`（网页的台前舞台）
        
        - 🏗️ **内容骨架**：`<h1>`~`<h6>`（大标题）、`<p>`（文本段落）。
            
        - 📊 **数据排列**：`<ul>` 嵌套 `<li>`（列表项）。
            
        - 🔗 **多媒体与交互**：`<a>`（负责跳走）、`<img>`（负责看）、`<button>`（负责点）、`<input>`（负责打字）。
            
        - 📦 **现代大管家**：`<div>`。
