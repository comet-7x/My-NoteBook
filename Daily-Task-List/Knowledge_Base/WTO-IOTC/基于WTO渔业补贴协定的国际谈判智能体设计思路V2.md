# 基于 WTO 渔业补贴协定的国际谈判智能体设计思路（完善版）

## 项目目标

本项目并不是模拟一场 **WTO 会议（WTO Meeting）**，而是模拟 **WTO 成员之间围绕渔业补贴展开的利益博弈（Interest-based Negotiation）**。

学生真正学习的对象不是协定本身，而是：

- 为什么不同国家会提出不同立场？
- 为什么这些国家长期坚持自己的观点？
- 为什么有些国家愿意妥协，而有些国家宁可谈判破裂？
- 国家利益、环境保护、经济发展之间是如何平衡的？

因此，本项目最终目标是培养学生：国际谈判能力、国家利益分析能力、政策分析能力、联盟构建能力、利益交换（Trade-off）能力。

整体流程分为三段：**课前 → 课堂 → 课后**。老师负责课堂；AI 负责课堂前后的能力培养（以及课前的"预演谈判"陪练，见后文第五节）。

---

## 一、什么是 WTO 渔业补贴协定（Agreement on Fisheries Subsidies）

### 1. 背景

世界上许多国家都会对本国渔业提供补贴，例如：中国的燃油补贴、日本的远洋捕捞船队补贴、欧盟的渔港与燃油相关补贴、韩国的大型捕捞企业补贴、美国的渔船现代化与设备补贴。

这些补贴虽然能提高渔业竞争力，但会带来全球性的资源问题：

```
捕鱼成本下降 → 更多渔船出海 → 捕捞能力持续增加 → 鱼类资源下降 → 过度捕捞 → 全球渔业资源衰退
```

据 FAO，全球约 38% 的已评估海洋鱼类种群处于过度捕捞状态。因此，联合国可持续发展目标 **SDG 14.6** 提出：到 2020 年前取消导致过度捕捞和非法捕鱼的有害渔业补贴（该时间目标已错过，谈判随之延续）。

### 2. WTO 谈判时间线

|时间|事件|
|---|---|
|2001|多哈回合正式启动渔业补贴谈判|
|2015|SDG 14.6 提出取消有害渔业补贴|
|2022.06|MC12 通过第一阶段协定（Fish 1）|
|2024.02–03|MC13（阿布扎比）就 Fish 2 极度接近达成，但最终未成|
|**2025.09.15**|**协定正式生效**（三分之二成员批准；到 2026 年 5 月已有 119 个成员批准）|
|2026.03|MC14（喀麦隆·雅温得）：Fish 2 仍未谈成，目标推至 MC15|
|**约 2029.09**|**日落条款截止点**：若届时 Fish 2 全面纪律仍未通过，整个协定（含 Fish 1）自动终止，除非总理事会另有决定|

> **教学用法提示**：日落条款是天然的"倒计时"。它解释了"为什么有些国家愿意妥协、有些宁可破裂"——因为破裂的代价是大家一起失去协定。建议在模拟里把它设为显式的截止期压力。

## 二、目前协定主要谈什么？

### 第一阶段（Fish 1，已完成并生效）

包含三项禁止，争议相对较小：

1. **禁止对 IUU（非法、未报告、无管制）捕鱼提供补贴**：企业若被认定从事 IUU 捕鱼，政府不得继续补贴。
2. **禁止对已枯竭鱼群继续补贴**：某鱼种资源严重减少时，不得再补贴其捕捞（如部分金枪鱼、鳕鱼、深海鱼）。
3. **禁止在无管理公海提供捕鱼补贴**：某片公海若无区域渔业管理组织（RFMO）监管，不得补贴企业前往捕鱼。

### 第二阶段（Fish 2，仍在谈判——学生模拟的主战场）

核心争议是 **捕捞能力补贴（Capacity Subsidy）**：新建/更新渔船、燃油补贴、税收优惠、贷款优惠、渔业设备补贴等。它牵涉各国产业政策，争议最大，最适合作为学生模拟谈判的主题。

**把它拆成可谈判的"议题清单"**：

|议题|选项区间|大致倾向|
|---|---|---|
|**① 能力补贴是否设硬性上限（cap）？**|A 硬性封顶并逐步削减 / B 有条件豁免（管理良好的资源可继续） / C 不设上限、仅要求透明|太平洋岛国、（部分）美国倾向 A；欧盟、日韩倾向 B/C|
|**② 单一层级 vs 两层级？**|A 单一层级（所有人同一规则） / B 两层级（大补贴国更严、低收入国更宽）|美国倾向 A；印度、发展中国家倾向 B|
|**③ 发展中国家过渡期（S&DT）多长、多宽？**|短（约 7 年）/ 中（约 15 年）/ 长（约 25 年）；是否永久豁免小规模/手工渔业|印度、LDC 要长且宽；美国要窄；中国"发展中身份"在此成为争点|
|**④ 燃油补贴是否纳入禁止范围？**|纳入 / 软化 / 排除|改革派想纳入；欧盟、日韩、中国敏感|
|**⑤ 远洋捕捞（distant-water）补贴如何约束？**|严管 / 一般义务 / 基本不管|沿海国、岛国、（近期）美国、印度想严管；远洋大国想宽|

> 每个学生小组代表一个国家，针对五项谈判议题都有对应的诉求立场，仅能从中挑选一到两项划定绝对不让步的底线；所有议题合并为一份完整协议文本，不存在分项表决，只有全体国家全部同意整套文件，本次谈判才算成功落地。

## 三、模拟谈判的核心思想

这门课不是"学习 WTO 法律"，而是"理解不同国家为什么坚持不同立场"。

- 国家没有"正确答案"；每个国家都站在自身利益角度谈判。
- 捕鱼大国希望保留补贴；环保导向国家希望限制补贴；发展中国家希望获得更多发展时间；小岛国希望保护渔业资源。
- **谈判的本质是利益协调，而不是观点辩论。**

**两条强化设定：**

- **协商一致（consensus）**：WTO 不投票表决，任何一个成员都能否决。所以学生无法靠"人多"取胜，必须搭联盟、做交换——这正是本课要训练的能力。
- **破裂的后果（BATNA）**：谈不成不是"没损失"。在日落条款下，长期谈不成意味着连 Fish 1 也一起失效。这让"宁可破裂"成为一个有分量的真实选择，而非免费选项。

## 四、课程设计（六阶段学习旅程）

```
认识问题 → 认识自己的国家 → 认识其他国家 → 制定谈判策略 → 正式谈判 → 复盘分析
```

### 第一阶段：认识问题（Understand the Problem）

**目标**：理解世界为什么要谈渔业补贴。

```
政府为什么补贴？ → 补贴有哪些？ → 为什么会导致过度捕捞？ → 为什么 WTO 要管？
```

**AI 教学内容**（逐条讲解相关条款）：一句话解释 → 为什么制定 → 哪些国家支持 → 哪些国家反对 → 历史争议是什么。建议直接锚定在 Fish 2 的"能力补贴"上，因为那才是当下真正在争的活议题。

### 第二阶段：认识自己的国家（Know Yourself）

学生随机或自主选择一个国家，AI 生成完整**国家画像**：

- **基本情况**：人口、GDP、人均 GDP、主要产业、海岸线长度、渔业就业人口。
- **渔业情况**：捕鱼量、水产品出口量、主要鱼种、远洋捕鱼规模、国内消费。
- **渔业补贴情况**：补贴规模、类型、历史变化。
- **国内政治压力**：渔民协会（要求保留补贴）、环保组织（要求减少远洋捕捞）、财政部门（希望减支）、地方政府（希望保就业）。

> **数据来源与防幻觉**：以上数字应对照真实数据，不能由模型自由编造。可能来源：FAO《世界渔业和水产养殖状况》(SOFIA)、OECD 渔业支持估算 (FSE) 数据库、WTO 成员官方补贴通报。这部分数据需要提前确定存入知识库，后续可以在平台手动更新

### 第三阶段：认识其他国家（Know Others）

重点不是"别人说过什么"，而是"为什么这样说"。

**国家立场分类**：

- **第一类·大型捕鱼国**：中国、日本、韩国、欧盟、中国台湾。诉求：不要立即取消所有补贴。
- **第二类·发达国家（环保/监管导向）**：美国、加拿大、澳大利亚、新西兰。关注：环保、监管、透明度、IUU 治理。
- **第三类·发展中国家**：印度、印度尼西亚、菲律宾、斯里兰卡。诉求：特殊与差别待遇（S&DT），要更多发展时间。
- **第四类·最不发达国家（LDC）**：孟加拉国、坦桑尼亚、莫桑比克。诉求：更多豁免、能力建设与技术援助。
- **第五类·小岛国（SIDS）**：马尔代夫、斐济、萨摩亚。诉求：保护渔业资源。

> **两个注意要点**：
> 1. **美国 vs 欧盟**：两者都是补贴大户，却谈不到一起，美国要"一刀切的严规则 + 严管远洋"，欧盟要保住自己的补贴（尤其燃油）。
> 2. **中国的双重身份**：既是最大补贴国、又主张发展中国家待遇，这是整个谈判最核心的争点，建议单独做成案例。

**国家历史立场**（展示 2017 年至今的关键观点）——**此项必须人工校准，不可由 AI 自由生成**，否则易出现错误立场。示例：印度长期坚持保护小规模渔民、要求长期 S&DT；中国支持发展中国家待遇与资源保护、反对简单取消全部补贴；欧盟支持透明度与可持续捕捞；美国支持限制有害补贴、强调 IUU 治理（近年更倾向单一层级、并对远洋补贴趋严）。

**联盟关系图**：AI 分析潜在盟友、潜在竞争者、摇摆国、可争取的国家，帮助学生形成外交思维。

### 第四阶段：制定谈判策略（Build Strategy）

正式谈判前，AI 协助学生生成本国战略：
1. **核心目标**（例：保护燃油补贴）。
2. **红线**（例：补贴削减绝不超过 30%）。
3. **可交换利益**（例：可接受"提高补贴透明度"，换取"保留远洋捕鱼补贴"）。
4. **谈判对象**（优先联系哪些国家、为什么）。
5. **风险分析**（例：欧盟可能提高监管要求；美国可能要求进一步削减）。

学生最终产出：**谈判简报（Negotiation Brief）**。例如：简报明确写出本国在第二节第五条议题上的立场与红线，方便进入正式谈判。

### 第五阶段：正式谈判（Negotiation）

尽量模拟真实 WTO 流程（注意：以**协商一致**作为谈判收尾，而非投票）：

```
主席宣布会议开始 → 各国开场陈述 → 自由磋商 → 形成联盟 → 提出修正案（对议题文本加/去方括号）
→ 再次磋商 → 形成最终文本 → 协商一致通过（无人正式反对即通过） → 会议结束
```

- **文本以"方括号"表示争议**：有争议的措辞先放进方括号，谈拢一条就去掉一条括号。
- **截止期压力**：引入日落倒计时——到点仍未达成，则本轮视为破裂，触发复盘中的"破裂后果"分析。
- **AI 负责**：记录发言、更新文本、判断程序是否合法（例如是否真的达成"无人反对"的一致）。

### 第六阶段：谈判复盘（Reflection）

AI 自动生成分析报告：

- **国家目标完成率**：例 ✔ 保留燃油补贴 / ✔ 获得 S&DT / ✘ 未能扩大捕鱼配额。
- **谈判行为分析**：主动发言、联盟建立、妥协、提出修正案等次数（**仅作过程参考，不直接用于打分**）。
- **与历史立场比较**：例——历史上的印度始终坚持 S&DT；本次学生最终放弃 → AI 指出此次谈判偏离历史立场。
- **改进建议**：例——若重来，建议先建联盟、再提修正案、最后做利益交换。

**评分标准（rubric，新增）**——建议按能力维度评，各 20%：

|维度|看什么|
|---|---|
|立场分析准确度|是否准确把握本国经济/政治/社会利益，立场是否站得住|
|策略与红线|谈判简报是否清晰，红线是否合理并守住|
|联盟与外交|是否有效识别并争取盟友、处理摇摆国|
|利益交换质量|让步是否换来对等收益，而非无谓妥协或一味僵硬|
|结果与复盘|是否达成本国可接受的可行协议（或有理由地选择破裂）；复盘是否深入、能与历史立场对照|

> 不建议按"发言次数/修正案次数"直接打分——那会奖励数量而非质量。

## 五、（新增）课前 AI 预演谈判（Rehearsal）

在正式课堂之前，让学生先与 2–3 个由 AI 扮演的其他国家做一次**小规模预演**。价值：

- 在压力下检验策略，提前发现自己真实的红线；
- 练习联盟试探与利益交换；
- 带着准备好的方案进课堂，课堂时间用于更高质量的真人博弈。

这一步正是"智能体应用"技术优势的用武之地：AI 依据第三阶段校准过的各国立场进行扮演，给学生一个安全的练习场。

## 六、（新增）AI 的四类角色与数据要求

同一个 app 里，AI 承担四种不同角色，建议在实现上分清：

1. **导师（Tutor）**：阶段 1–4，讲解条款、生成国家画像与策略。
2. **陪练对手（Sparring Partner）**：课前预演里扮演他国。
3. **主持人/记录员（Facilitator）**：阶段 5，记录发言、维护带方括号的谈判文本、判断"协商一致"是否真正达成、管理日落倒计时。
4. **分析师（Analyst）**：阶段 6，生成复盘与评分。

**数据与防幻觉总原则**：凡涉及事实（国别数字、历史立场）一律以真实数据/人工校准为准，模型不得自由生成；不确定处标注为"示意性估算"。

## 七、教学目标

学生完成整个课程后，应具备：

1. 理解 WTO 渔业补贴协定的背景与核心争议；
2. 能分析不同国家立场形成的经济、政治和社会原因；
3. 能基于国家利益制定谈判目标、红线和利益交换方案；
4. 能在动态环境下进行联盟构建、文本协商和策略调整；
5. 能复盘谈判过程，并与历史谈判实践进行比较分析。

最终，学生学习的不仅是一项国际协定，更是国际经贸谈判中的利益分析、政策制定和协商决策能力。

---

## 八、建议的分阶段落地顺序（给初学者的 build order）

1. **先定"谁在谈判桌 + 各方要什么"**：小规模从 6 方起步（欧盟、美国、中国、印度、太平洋岛国、LDC 集团）。
2. **把 6 方各写成一页"授权书（mandate）"**：包含核心目标、红线、可交换项、在五条议题上的立场——让 AI 智能体能照着"演"。
3. **写小型谈判文本**：把第二节的五条议题写成带方括号的条款草案。
4. **设回合结构与规则**：开场陈述 → 磋商 → 修正案 → 协商一致通过；加入日落倒计时。
5. **最后接复盘与评分**：按第六阶段的 rubric 生成报告。
