# 一、课程基本信息

课程名称：
1. 中文名称：国际渔业管理与法规
2. 英文名称：International fisheries management

适用专业：海洋管理
授课对象：海洋类研究生

先修课程及要求：本课程的前修课程为《海洋学概论》和《国际海洋法》。在掌握海洋学的基本知识和国际海洋法的基本框架之后，学生将进一步学习国际渔业法的理论与制度，理解其在全球海洋治理中的地位与作用。

# 二、课程性质与目的

本课程是上海海洋大学法学硕士研究生的专业学位课。本课程通过教与学，使学生掌握国际渔业管理的基本理论、法律框架、主要制度及实践应用，培养学生在全球化背景下处理渔业管理和法律问题的能力，具体包括以下几个方面：（1）使学生理解国际渔业管理的重要性和复杂性，以及国际社会普遍认可的渔业资源可持续利用原则、渔业管理制度和措施；（2）使学生认识国际渔业管理的生物和生态学、社会和经济维度，理解科学对渔业管理决策的重要支撑作用；（3）使学生系统掌握渔业管理的主要国际条约框架、重要的非法律性文件的主要内容及其在国际渔业管理中的作用；（4）增强学生的国际视野，提升国际合作意识。

# 二、课程简介

本课程围绕渔业资源养护与管理主题，主要讲授国际渔业管理系统框架和主要组成内容，特别是国际渔业管理的条约和相关文书、制度框架、管理措施、决策方法、科学信息在管理中的应用等关键内容。通过理论讲授、案例分析、小组讨论等多种教学方式，使学生能够全面理解和掌握国际渔业管理的法律框架、制度措施和运行机制。

# 四、课程考核评价方式

课程教学主要采用理论讲授、课堂讨论、学生专题报告、课后自学等教学方法，综合利用多媒体、在线课程平台等信息化教学辅助手段，并通过提供教学参考资料、推荐课外阅读材料等方式深化学生对课程知识的理解和掌握程度，拓宽学生的知识面与知识结构。


## （一）考核方式

课程考核采用论文写作与课堂表现相结合的方式，重点考核学生对本课程所涉及的国际渔业法的基本概念、基本理论、制度框架和当代实践问题的掌握情况，以及运用国际渔业法原理与工具分析国际渔业治理案例的能力。

考核成绩由平时成绩和期末成绩构成，分别占 60% 和 40%。其中，平时成绩由小组汇报、课堂讨论、考勤与平时表现等部分构成；期末成绩采用小论文考核方式。

# 五、教学方法
1. 翻转课堂教学法：基于OBE理念，将课堂教学内容进行分解。第一阶段为上课前一周，教师发布学习资料及相关国际法律文件，由学生自主完成资料学习。第二阶段为授课的一周：在第一小节课上以提问方式要求学生根据所阅读资料进行回答，组织讨论；第二小节课针对学生的回答和讨论结果，教师进行补充与系统性梳理，讲授本节课的核心内容。第三阶段为课后的一周：学生完成教师课堂布置的作业，并自主学习下一周的内容。

2. 项目教学法：学生通过组队，围绕国际渔业治理的专题（如IUU捕鱼治理、渔业补贴谈判、RFMO管理案例等）开展调研，充分运用课堂所学知识，理解所学内容，并通过报告展示所学知识，更好地掌握和运用国际渔业法的理论与制度
