# 一、课程基本信息

课程名称：
1. 中文名称：国际渔业组织与全球治理
2. 英文名称：International Fisheries Organizations and Global Governance

适用专业：海洋管理
授课对象：海洋类研究生

先修课程及要求：本课程的前修课程为《海洋学概论》和《国际海洋法》。在掌握海洋学的基本知识和国际海洋法的基本框架之后，学生将进一步学习国际渔业法的理论与制度，理解其在全球海洋治理中的地位与作用。

# 二、课程性质与目的

本课程是上海海洋大学法学研究生的专业学位课。课程目的在于通过教与学，使学生能够运用相关概念、理论、原则和规则分析国际渔业重要议题，思考我国参与全球渔业治理的路径，具备以下能力：第一，了解国际规则的运行机制，具有国际视野；第二，掌握国际重要法律文书的具体内容；第三，了解我国参与全球渔业治理的基本立场和路径，能够就当前热点议题发表评论。

# 二、课程简介

本课程主要讲授与全球渔业治理相关的国际组织的基础理论与相关基本知识，课程内容主要包括渔业治理的基础理论、涉及渔业的国际组织的基本知识以及联合国、联合国粮农组织、区域渔业管理组织、国际海事组织、国际劳工组织、世界贸易组织等国际组织的组织结构、议事规则和所涉渔业事务及制度、措施体系，以及联合国可持续发展议程、打击IUU捕捞、负责任渔业、、公海保护区、渔业补贴、渔业海上安全等全球渔业议题发展进程。

# 四、教学基本要求

教师在课堂上采取多种方式讲解课程内容，一方面通过讲授详细介绍每章的重难点，另一方面要求学生通过专题报告等形式自主学习课程内容，通过课堂讨论引导学生将理论与实际想结合，启迪学生的思维，加深学生对知识的理解。


# 五、教学方法
教学主要以知识点讲授+学生专题报告+课堂互动+老师系统分析点评等方式进行，将课前辅导与翻转课堂模式相结合，学生专题报告与老师系统介绍知识相结合，激发学生主动学习。课程成绩由出勤、课堂讨论、小组报告和期末论文四部分组成。期末论文采用专题分析的方式进行，考察学生知识掌握和运用知识分析解决问题的能力。
