## 项目目标

本项目并不是模拟一场 **WTO 会议（WTO Meeting）**，而是模拟 **WTO 成员之间围绕渔业补贴展开的利益博弈（Interest-based Negotiation）**。

学生真正学习的对象不是协定本身，而是：

- 为什么不同国家会提出不同立场？
- 为什么这些国家长期坚持自己的观点？
- 为什么有些国家愿意妥协，而有些国家宁可谈判破裂？
- 国家利益、环境保护、经济发展之间是如何平衡的？

因此，本项目最终目标是培养学生：国际谈判能力、国家利益分析能力、政策分析能力、联盟构建能力、利益交换（Trade-off）能力。

整体流程分为三段：**课前 → 课堂 → 课后**。老师负责课堂组织；AI 负责课堂前后的能力培养。

## 一、什么是 WTO 渔业补贴协定（Agreement on Fisheries Subsidies）

### 1. 背景

世界上许多国家都会对本国渔业提供补贴，例如：中国的燃油补贴、日本的远洋捕捞船队补贴、欧盟的渔港与燃油相关补贴、韩国的大型捕捞企业补贴、美国的渔船现代化与设备补贴。

这些补贴虽然能提高渔业竞争力，但会带来全球性的资源问题：

```
捕鱼成本下降 → 更多渔船出海 → 捕捞能力持续增加 → 鱼类资源下降 → 过度捕捞 → 全球渔业资源衰退
```

据 FAO，全球约 38% 的已评估海洋鱼类种群处于过度捕捞状态。因此，联合国可持续发展目标 **SDG 14.6** 提出：到 2020 年前取消导致过度捕捞和非法捕鱼的有害渔业补贴（该时间目标已错过，谈判随之延续）。

### 2. WTO 谈判时间线

| 时间             | 事件                                                                          |
| -------------- | --------------------------------------------------------------------------- |
| 2001           | 多哈回合首次将渔业补贴纳入谈判范围                                                           |
| 2015           | SDG 14.6 要求，到2020年，禁止导致产能过剩和过度捕捞的渔业补贴，取消对非法、不报告、不管制（IUU）捕捞的补贴               |
| 2022.06        | MC12 通过第一阶段协定（Fish 1）                                                       |
| **2025.09.15** | **协定正式生效**（三分之二成员批准；到 2026 年 5 月已有 119 个成员批准）                               |
| 2026.03        | MC14（喀麦隆·雅温得）：Fish 2 仍未谈成，目标推至 MC15                                         |
| **约 2029.09**  | **日落条款截止点**：根据《渔业补贴协定》第12条，若届时 Fish 2 全面纪律仍未通过，除非总理事会另有决定，整个协定（含 Fish 1）将终止 |

> **教学用法提示**：日落条款是天然的"倒计时"。它解释了"为什么有些国家愿意妥协、有些宁可破裂"——因为破裂的代价是大家一起失去协定。建议在模拟里把它设为显式的截止期压力。

## 二、目前协定主要谈什么？

### 第一阶段（Fish 1，已完成并生效）

包含三项禁止，争议相对较小：

1. **禁止对从事或支持 IUU（非法、未报告、无管制）行为的船舶或经营者提供补贴**：船舶或经营者若被认定从事 IUU 捕鱼，政府不得继续补贴。
2. **禁止对过度捕捞鱼类种群的捕捞或与捕捞有关的活动提供补贴**：某鱼类种群进入过度捕捞状态时，不得对**捕捞或与捕捞有关的活动提供**补贴 。
3. **禁止对在无管辖公海从事捕捞或与捕捞有关的活动提供补贴**：在沿海非成员管辖范围外，并在区域渔业管理组织/安排范围外，捕捞或与捕捞有关的活动不得给予补贴。

**Fish1 阶段谈判的 5 项可磋商核心议题清单**（用于回顾"当年争过什么"）：

| 议题                         | 谈判可选方案区间                                                   | 各国当年大致立场                                         |
| -------------------------- | ---------------------------------------------------------- | ------------------------------------------------ |
| **① IUU 捕捞认定与处罚规则**        | A 简化认定流程、快速停发补贴 / B 增设多层信息交换、听证缓冲期 / C 仅 RFMO 认定有效，沿岸国无权判定 | 太平洋岛国、环保方支持 A；远洋渔业大国、发展中国家倾向 B；部分发达国家主张 C        |
| **② 枯竭鱼种补贴豁免边界**           | A 仅大规模种群修复项目可豁免 / B 小型生计渔业修复补贴全部豁免 / C 放宽标准，普通养护补贴均可豁免     | 欧美、岛国倾向 A；发展中国家、最不发达国家（LDC）支持 B                  |
| **③ 无监管公海条款适用范围**          | A 全部无 RFMO 公海一律禁补贴 / B 传统历史捕捞海域设置例外 / C 小型近海渔船远洋作业豁免       | 沿海小国、环保阵营选 A；拥有远洋船队的国家争取 B/C                     |
| **④ 发展中国家特殊差别待遇（S&DT）宽限期** | A 无缓冲、立即全面执行禁令 / B 2 年争端豁免缓冲期（最终落地版本） / C 5 年长缓冲、近海永久豁免    | 美国、欧盟倾向 A/B；印度、LDC、中国等发展中成员全力争取 C，最终折中达成 2 年缓冲条款 |
| **⑤ 透明度通报**                | A 各国每年完整上报全部渔业补贴 / B 低产量小国每 4 年上报、简化材料 / C 自愿通报、无强制义务      | 发达国家要求 A；小型渔业发展国、LDC 诉求 B；少数国家支持 C               |
| **⑥技术援助机制**                | A 发达国家提供技术援助/B技术援助资金由WTO或捕捞大国提供/ C严格控制技术援助受益方和技术援助范围       | 发展中国家和最不发达国家要求A和B，发达国家要求C                        |

> 每个学生小组代表一个国家，针对五项谈判议题都有对应的诉求立场，仅能从中挑选一到两项划定绝对不让步的底线；所有议题合并为一份完整协议文本，不存在分项表决，只有全体国家全部同意整套文件，本次谈判才算成功落地。

## 三、模拟谈判的核心思想

这门课不是"学习 WTO 法律"，而是"理解不同国家为什么坚持不同立场"。

- 国家没有"正确答案"；每个国家都站在自身利益角度谈判。
- 捕鱼大国希望保留补贴；环保导向国家希望限制补贴；发展中国家希望获得更多发展时间；小岛国希望保护渔业资源。
- **谈判的本质是利益协调，而不是观点辩论。**

**两条强化设定：**

- **协商一致（consensus）**：WTO 不投票表决，任何一个成员都能否决。所以学生无法靠"人多"取胜，必须建立联盟、做交换，这是本课要训练的能力。
- **破裂的后果（BATNA）**：谈不成不是"没损失"。在日落条款的约束下，长期谈不成意味着连 Fish 1 也一起失效。这让"宁可破裂"成为一个有分量的真实选择，而非免费选项。

## 四、课程总体设计
课程贯穿"课前 → 课堂 → 课后"三段，落到学习闭环上，即：
```
知识学习 → 国家准备 → 策略制定 → 课堂谈判 → AI 复盘
```
其中课前（认识问题、认识自己、认识他国、制定策略）与课后（复盘）由 AI 主导能力培养，课堂（正式谈判）由教师组织、AI 提供秘书处支持。展开为六个阶段（见下一节）。


## 五、六阶段学习流程

```
认识问题 → 认识自己的国家 → 认识其他国家 → 制定谈判策略 → 正式谈判 → 复盘分析
```

### 第一阶段：认识问题（Understand the Problem）

**目标**：理解世界为什么要谈渔业补贴。

```
政府为什么补贴？ → 补贴有哪些？ → 为什么会导致过度捕捞？ → 为什么 WTO 要管？
```

逐条讲解相关条款：一句话解释 → 为什么制定 → 哪些国家支持 → 哪些国家反对 → 历史争议是什么。

> **AI 支持**：讲解协定案文与条款背景，回答概念性问题，用例子解释"补贴如何导致过度捕捞"。 **教师负责**：把控进度，引导讨论，联系课程整体目标。

### 第二阶段：认识自己的国家（Know Yourself）

学生随机或自主选择一个国家，通过自己查找资料，了解所选国家的基本国情，具体范围包括但不仅涵盖以下内容：

- **基本情况**：人口、GDP、人均 GDP、主要产业、海岸线长度、渔业就业人口。
- **渔业情况**：捕鱼量、水产品出口量、主要鱼种、远洋捕鱼规模、国内消费。
- **渔业补贴情况**：补贴规模、类型、历史变化。
- **国内政治压力**：渔民协会（要求保留补贴）、环保组织（要求减少远洋捕捞）、财政部门（希望减支）、地方政府（希望保就业）。

> **数据来源与防幻觉**：以上数据应对照真实数据，不能由模型自由编造。可能来源：FAO《世界渔业和水产养殖状况》(SOFIA)、OECD 渔业支持估算 (FSE) 数据库、WTO 成员官方补贴通报。这部分数据需提前确定并存入知识库，后续可在平台手动更新。

> **AI 支持**：基于知识库生成国家画像，梳理该国的经济/政治/社会利益。（供老师参考验证）。
> **教师负责**：分配或确认各组代表的国家，提示学生"你代表的是整个国家，而非某一群体"。

### 第三阶段：认识其他成员（Know Others）

重点不是"别人说过什么"，而是"为什么这样说"。

**成员立场分类**：

- ● **第一类·渔业规模较大成员**：中国、日本、韩国、欧盟、台湾地区（台澎金马单独关税区）。诉求：不要立即取消所有补贴。
- **第二类·发达成员（环保/监管导向）**：美国、加拿大、澳大利亚、新西兰。关注：环保、监管、透明度、IUU 治理。
- **第三类·发展中成员**：印度、印度尼西亚、菲律宾、斯里兰卡。诉求：特殊与差别待遇（SDT，Special and Differential Treatment），要更多发展时间。
- **第四类·最不发达成员（LDC）**：孟加拉国、坦桑尼亚、莫桑比克。诉求：更多豁免、能力建设与技术援助。


**国家历史立场**（展示 2017 年至今的关键观点），**此项必须人工校准，不可由 AI 自由生成**，否则易出现错误立场。示例：印度长期坚持保护小规模渔民、要求长期 SDT；中国支持发展中国家待遇与资源保护、反对简单取消全部补贴；欧盟支持透明度与可持续捕捞；美国支持限制有害补贴、强调 IUU 治理（近年更倾向单一层级、并对远洋补贴趋严）。

**联盟关系图**：AI 分析潜在盟友、潜在竞争者、摇摆国、可争取的国家，帮助学生形成外交思维。

> **AI 支持**：提供各国（校准过的）历史立场，分析利益异同，生成联盟关系图与"可争取对象"。 **教师负责**：组织跨组交流，纠正明显误读，引导学生从"辩对错"转向"看利益"。


### 第四阶段：制定谈判策略（Build Strategy）

正式谈判前，AI 协助学生生成本国战略：
1. **核心目标**（例：保护燃油补贴）。
2. **红线**（例：补贴削减绝不超过 30%）。
3. **可交换利益**（例：可接受"提高补贴透明度"，换取"保留远洋捕鱼补贴"）。
4. **谈判对象**（优先联系哪些国家、为什么）。
5. **风险分析**（例：欧盟可能提高监管要求；美国可能要求进一步削减）。

学生最终产出：**谈判简报（Negotiation Brief）**。例如：简报明确写出本国在第二节第五条议题上的立场与红线，方便进入正式谈判。

> **AI 支持**：作为"知识库助手"提出目标、红线、可交换项、联盟与风险的建议草案。
> **教师负责**：审阅简报是否合理，提醒红线不宜过多（否则无从交换）。**决策权始终在学生手里。**

### 第五阶段：正式谈判（Negotiation）

尽量模拟真实 WTO 流程（注意：以**协商一致**作为谈判收尾，而非投票）：

```
主席宣布会议开始 → 介绍本次会议议题 → 各国举手发表意见 → 形成联盟 → 提出修正案（对议题文本内容增加或者删除，或更改部分表述）→ 再次发表各自意见 → 形成最终文本 → 表决一致通过 → 会议结束
```

- **截止期压力**：引入日落倒计时，到点仍未达成，则本轮视为破裂，触发复盘中的"破裂后果"分析。

>**AI 支持（谈判秘书处角色）**：实时记录发言、维护带标记的谈判文本、判断程序是否合法（如是否真正达成"无人反对"的一致）、管理倒计时。
 **教师负责**：担任/指定主席，主持流程，掌控节奏，处理僵局。

### 第六阶段：谈判复盘（Reflection）

AI 自动生成分析报告：

- **国家目标完成率**：例 ✔ 保留燃油补贴 / ✔ 获得 SDT / ✘ 未能扩大捕鱼配额。
- **谈判行为分析**：主动发言、联盟建立、妥协、提出修正案等次数（**仅作过程参考，不直接用于打分**）。
- **与历史立场比较**：例——历史上的印度始终坚持 SDT；本次学生最终放弃 → AI 指出此次谈判偏离历史立场。
- **改进建议**：如果全部重来，建议先建联盟、再提修正案、最后做利益交换。

**评分标准**——按能力维度评，各 20%：

|维度|看什么|
|---|---|
|立场分析准确度|是否准确把握本国经济/政治/社会利益，立场是否站得住|
|策略与红线|谈判简报是否清晰，红线是否合理并守住|
|联盟与外交|是否有效识别并争取盟友、处理摇摆国|
|利益交换质量|让步是否换来对等收益，而非无谓妥协或一味僵硬|
|结果与复盘|是否达成本国可接受的可行协议（或有理由地选择破裂）；复盘是否深入、能与历史立场对照|
>**AI 支持**：自动生成复盘报告与评分建议，给出可操作的改进建议。 
>**教师负责**：结合课堂观察给出最终评价（AI 评分仅作参考）。

## 六、本平台与课堂教学流程的对应关系

本平台并不替代课堂模拟，而是在教师原有教学流程基础上提供 AI 辅助。对应关系如下：

| 教师课堂流程      | AI 辅助内容                       |
| ----------- | ----------------------------- |
| 阅读协定案文（W20） | 讲解案文、解释条款背景                   |
| 分析国家立场      | 生成国家画像、历史立场分析                 |
| 提交谈判方案      | 协助生成谈判简报                      |
| 正式谈判        | 担任国家知识库助手与谈判秘书处（记录、维护文本、程序判断） |
| 课后总结        | 生成谈判复盘报告                      |

因此：课堂仍由教师组织；AI 负责提升学生课前准备质量与课后学习效果。**AI 不是来替代老师，而是辅助老师。**


## 七、AI 支持内容与分工

**AI 主要负责**：

- ● 自动生成国家画像**供老师校对学生准备工作**；
- 提供（人工校准过的）国家历史立场；
- 分析国家利益、推荐潜在联盟；
- 协助制定谈判策略；
- 实时记录谈判过程、维护文本；
- 谈判结束生成复盘报告与评分建议。

**教师主要负责**：

- 控制课堂节奏；
- 主持谈判流程；
- 回答课堂问题；
- 最终评价学生表现。

为便于开发，AI 的工作可归为四种角色：**知识导师**（阶段一，讲解）、**国家智库 / Policy Advisor**（阶段二至四，分析与建议）、**谈判秘书处**（阶段五，记录与程序）、**复盘分析师**（阶段六，复盘与评分）。


## 八、教学目标

学生完成整个课程后，应具备：

1. 理解 WTO 渔业补贴协定的背景与核心争议；
2. 能分析不同国家立场形成的经济、政治和社会原因；
3. 能基于国家利益制定谈判目标、红线和利益交换方案；
4. 能在动态环境下进行联盟构建、文本协商和策略调整；
5. 能复盘谈判过程，并与历史谈判实践进行比较分析。

最终，学生学习的不仅是一项国际协定，更是国际经贸谈判中的利益分析、政策制定和协商决策能力。


## 九、系统边界

本平台**主要提供**：知识学习、国家分析、谈判策略建议、谈判记录、复盘分析。

本平台**不负责**：
- 自动替学生完成谈判；
- 自动生成"最终谈判答案"；
- 替代学生进行决策；
- 替代教师进行课堂评价。

AI 仅作为**智库**工具，所有最终决策均由学生自主完成。明确这条边界，教师用起来更放心，也守住了本课"训练学生自己判断"的初衷。
