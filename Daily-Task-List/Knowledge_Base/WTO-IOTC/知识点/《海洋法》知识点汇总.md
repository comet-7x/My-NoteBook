# 一、课程培养目标对应核心知识框架
## 课程目标1：系统掌握海洋法基础理论与全域海域法律制度，夯实专业基础知识
1. 海洋法历史发展脉络、国际统一编撰全过程、关键国际条约演进
2. 《联合国海洋法公约》（UNCLOS）立法背景、谈判历程、整体制度框架
3. 九大海洋区域法定定义、边界划分标准、基础法律地位：内水、领海、毗连区、群岛水域、国际航行海峡、专属经济区、大陆架、公海、国际海底区域
4. 海洋环境保护、海洋科学研究通用基础国际法规则

## 课程目标2：具备法理分析能力，实现理论向实务问题转化
1. 透过海洋争端表象提炼法律争议焦点的分析方法
2. 不同海域制度交叉冲突场景下法律适用判断逻辑
3. 海洋划界、船舶航行、资源开发、海洋污染等实务问题的法律定性思路
4. 国际海洋司法、海洋时事案例拆解与论证框架

## 课程目标3：掌握海洋实务解决方案，立足国际法维护国家海洋权益
1. 沿海国各项海洋主权、主权权利、管辖权法定边界
2. 海洋权益争端有理有据的法律论证逻辑与方案设计思路
3. 公海捕捞、远洋船舶管控、深海资源开发等实务维权路径
4. 中日东海大陆架、南海群岛等我国周边海域争端法理依据与维权路径

## 课程目标4：培育团队协作职业素养，适配海洋管理岗位协同工作需求
1. 海洋法小组案例研讨、时事议题分工协作模式
2. 小组汇报分工、观点整合、分歧协调组织方法
3. 海洋管理、渔业管理岗位职业道德规范
4. 口头汇报、专业论文文字表达规范，兼顾团队成员与负责人双重角色能力

## 课程目标5：树立海洋强国意识，兼具家国情怀与综合人文素养
1. 海洋强国战略法治实施路径、国家海洋权益顶层价值
2. 我国领海基线、南海断续线、琼州海峡等制度背后主权与历史意义
3. 海权意识、爱国护海情怀、维护国家海洋权益使命感
4. 海洋命运共同体、生态文明建设与海洋法治融合内涵

# 二、分章节完整知识点（依据教学内容）
## 第一章 海洋法概述（4学时）
### 基础核心知识点
1. 海洋法起源、古代/近代/现代海洋法三阶段划分
2. 近代海洋自由论、闭海论核心理论与历史冲突
3. 海洋法国际编撰关键会议、条约沿革，从日内瓦四公约到UNCLOS
4. 《联合国海洋法公约》谈判、签署、生效全流程、核心定位
### 重难点
1. 世界海洋法发展阶段清晰划分标准
2. UNCLOS编撰过程中的各国分歧、条款妥协逻辑
### 思政配套知识点
全球海洋规则演变中发展中国家海洋权益诉求演进，我国参与国际海洋立法历史

## 第二章 内水（6学时）
### 基础核心知识点
1. 领海基线两大类型：正常基线、直线基线法定划定规则、适用场景
2. 内水法律定义、完全主权法律地位，与领海权利差异
3. 我国官方公布领海基线范围、划定标准
4. 我国渤海、内海湾、内河入海口等典型内水域管控制度
### 重难点
1. 两种领海基线区分适用边界
2. 内水排他性主权完整内涵
### 思政配套知识点
我国领海基线制度对近海国土主权、近海管控的战略保障作用

## 第三章 领海和毗连区（6学时）
### 基础核心知识点
1. 领海12海里法定宽度、外部界限测算规则
2. 沿海国领海完整主权内容、属地管辖全覆盖
3. 无害通过权定义、适用船舶、禁止行为清单、行使限制
4. 毗连区范围、海关/财政/移民/卫生四类管制权限
5. 中国《领海及毗连区法》国内立法核心条款
### 重难点
1. 领海完全主权与公海自由原则核心区分
2. 无害通过制度争议场景判定标准
### 思政配套知识点
我国领海、毗连区划定对近海国土安全、渔业资源管控的战略价值

## 第四章 群岛水域（4学时）
### 基础核心知识点
1. 群岛国、群岛基线法定构成条件
2. 群岛水域双重法律地位：主权覆盖+通行权利保留
3. 群岛海道通过权适用主体、权利义务规则
4. 群岛水域与领海、专属经济区边界区分标准
### 重难点
1. 群岛水域特殊双重法律属性
2. 群岛海道通过与领海无害通过权利差异
### 思政配套知识点
南海四大群岛法律定性、南海断续国界线历史法理双重价值

## 第五章 用于国际航行的海峡（4学时）
### 基础核心知识点
1. 国际航行海峡判定标准、三类海峡分类体系
2. 过境通行权完整规则，船舶、航空器权利义务
3. 琼州海峡专属特殊制度：中国排他内水管控模式
### 重难点
1. 不同类型海峡差异化通行制度适用边界
2. 过境通行、无害通过、完全禁止通行场景区分
### 思政配套知识点
琼州海峡专属法律地位，我国对内水海峡完整主权管控依据

## 第六章 专属经济区和大陆架（6学时）
### 基础核心知识点
1. 专属经济区200海里范围、非完全主权的主权权利属性
2. 沿海国资源开发、海洋科研、海洋环保专属管辖权
3. 大陆架自然延伸原则、200海里外扩展规则、外大陆架申请机制
4. 专属经济区与大陆架权利重叠、权责区分条款
5. 海洋划界公平原则、等距离中间线规则适用限制
### 重难点
1. 两大海域资源管理完整法律制度体系
2. 大陆架划界国际争端核心法理分歧
### 思政配套知识点
UNCLOS专属经济区规则对发展中沿海国海洋资源权益的保护意义

## 第七章 公海（6学时）
### 基础核心知识点
1. 公海定义、无主权归属法律地位
2. 六大公海自由完整内容、各项自由法定限制性条款
3. 船旗国专属管辖核心原则、船舶登记规则
4. 登临权、紧追权行使条件、法定流程、限制边界
5. 公海渔业资源养护国际通用规则
### 重难点
1. 公海自由原则边界约束体系
2. 船旗国管辖、普遍管辖并行适用场景区分
### 思政配套知识点
我国远洋公海捕捞权益保护、海外船舶公海维权法律路径

## 第八章 国际海底区域（4学时）
### 基础核心知识点
1. 国际海底区域“人类共同继承财产”核心法理原则
2. 平行开发制度设计逻辑、运作模式
3. 国际海底管理局组织架构、矿区申请、资源分配规则
4. UNCLOS第十一部分修改背景、核心条款调整内容
### 重难点
1. 人类共同继承财产内涵，与大陆架自然资源权属区分
2. 平行开发制度中发达国家与发展中国家利益平衡机制
### 思政配套知识点
中国国际海底矿区申请、深海资源勘探开发战略布局与海洋权益

## 第九章 海洋环境保护和海洋科学研究（4学时）
### 基础核心知识点
1. 海洋污染来源分类、全球海洋环保公约框架体系
2. 内水/领海/专属经济区/公海多层级污染管辖权划分
3. 海洋科学研究许可制度、沿海国审批管控权限
4. 中国海洋环保国内立法、近海生态治理体系
### 重难点
1. 跨海域跨国污染多层级管辖分工协调机制
2. 我国海洋环保治理体系与国际规则对接落地路径
### 思政配套知识点
我国海洋生态修复、污染治理、远洋海洋科考战略成果

## 第十章 中国的专属经济区和大陆架（4学时）
### 基础核心知识点
1. 中国专属经济区、大陆架国内法完整规定
2. 我国近海海域资源主权权利、管辖权覆盖范围
3. 周边国家海域划界冲突现状梳理
4. 中日东海大陆架划界争议核心法理分歧点
### 重难点
1. 我国海域权益主张国际法+国内法双重依据整合
2. 自然延伸原则与等距离中间线两种划界理论对抗
### 思政配套知识点
东海大陆架争端中我国基于国际法的合法维权逻辑与国家海洋战略立场

## 综合实践模块（全章节配套课堂讨论、小组汇报）
1. 单海域法律制度对比分析研究方法
2. 周边海洋争端多法条、多海域制度交叉论证框架
3. 海洋强国视角下海域权益维护对策撰写思路

# 三、参考书目与线上线下资料衍生拓展知识点
## 一、线上官方平台衍生知识点
1. 联合国官网
    - UNCLOS完整条约原文、联合国海洋法法庭判例
    - 全球海洋治理联大决议、海洋权益相关国际会议文件
2. 自然资源部官网
    - 中国领海基线公告、海岛保护、海域使用权法规
    - 全国海洋发展战略、年度海洋发展报告官方数据
3. 农业农村部渔业渔政管理局
    - 国际渔业养护协定、远洋渔船管控规则
    - 公海捕捞、IUU捕捞国内配套管理政策

## 二、线下外文经典教材拓展知识点
1. Donald R. Rothwell. The International Law of the Sea
    - 欧美主流海洋法理论、国际海洋争端司法裁判逻辑
    - 各国海域制度对比、公海与深海前沿法律研究
2. Yoshifumi Tanaka. The International Law of the Sea
    - 比较法视角海洋法体系、发展中国家海洋权益理论
    - 海洋划界、海洋环境领域国际判例深度解析

## 三、线下国内中文教材拓展知识点
1. 王铁崖《国际法》
    - 海洋法在整体国际法体系中的定位、主权与管辖权基础法理
2. 傅崐成《海洋法相关公约及中英文索引》
    - UNCLOS条文中英释义、条约冲突适用规则、专业术语精准界定
3. 屈广清、曲波《海洋法》
    - 本科适配系统化海域制度、国内配套立法、行政管理实务应用
4. 贾宇《海洋发展战略时论》
    - 海洋强国战略理论、地缘海权、海洋法治顶层设计
5. 《中国海洋发展报告》
    - 年度海洋实务、海域争端时事、国内海洋管理落地政策

# 四、教学考核对应的综合能力知识点
## 1. 小组专题报告能力要点
1. 课前自主研读UNCLOS公约、官方海洋政策、中外专著文献方法
2. 单海域完整分析框架：法律地位—核心制度—国内配套法规—实务争端—国家维权路径
3. 海洋热点议题多维分析视角：法律条文+国家战略+思政价值
4. 外文海洋法律文本阅读、核心观点提炼与口头汇报表达能力

## 2. 课堂讨论思辨能力要点
1. 辨析不同海域制度规则差异、制度适用冲突场景
2. 结合我国立场评析东海、南海、公海开发等海洋热点争端
3. 独立设计维护我国海洋权益的法治化完善方案

## 3. 期末结课论文综合能力要点
1. 国际公约、国内海洋法规、学术专著规范引用写作方法
2. 多海域、多条约交叉制度对比研究写作框架
3. 立足海洋强国战略视角，分析我国海洋权益法治保障路径
4. 结合案例评析现有海洋法制度缺陷与完善对策

## 4. 综合考核构成对应能力
- 课堂出勤+课堂讨论：熟练掌握基础海域制度，具备独立法理思辨能力
- 小组汇报（15%）：综合运用海域法律知识完成专题调研展示，具备团队协作能力
- 案例分析（10%）：运用海洋法条文拆解实务海洋争端
- 课后作业（10%）：巩固公约条文、国内海洋法规基础识记与简单应用
- 期末论文（60%）：整合全部课程知识，完成完整规范的海洋法律专题分析成果

## 5. 平时成绩、期末论文分层评分核心考点
1. 平时考核：各海域基础概念、小组协作表现、思政海洋强国认知、简单案例法理应用
2. 期末论文考核：多制度综合分析、海洋争端法理论证、国家海洋战略融合、海权爱国价值阐释
